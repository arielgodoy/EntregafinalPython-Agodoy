Información Técnica de la API
Endpoint base
/api/v1/maestros/rubros/
Métodos soportados
GET
POST
PUT
DELETE
Autenticación

La API usa sesión autenticada del sistema (cookie sessionid). Para métodos POST/PUT/DELETE se requiere CSRF token. No se utiliza API key externa.

Headers requeridos
Accept: application/json
X-Requested-With: XMLHttpRequest
X-CSRFToken: <csrftoken>
Contexto de empresa

El sistema es multiempresa: la empresa activa se obtiene desde la sesión del usuario. empresa_id se resuelve internamente en el backend. Un cliente externo debe autenticarse primero en el sistema.

Parámetros del recurso Rubro
codigo (string)
nombre (string)
ip (string)
localmatriz (string)
apostrofe (string)
colacion (boolean)
